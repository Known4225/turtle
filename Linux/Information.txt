Compile GLFW to binary: https://www.glfw.org/docs/3.3/compile.html#:~:text=GLFW%20uses%20CMake%20to%20generate,download%20CMake%20from%20their%20site
sudo apt install xorg-dev
sudo apt install libwayland-dev libxkbcommon-dev wayland-protocols extra-cmake-modules
sudo apt install cmake
cmake -S . -B build
cd build
make
the binary will be glfw-3.3.8/src/libglfw3.a

Compile glad to binary: gcc glad.c -O3 -I./../include -c -o libglad.a
